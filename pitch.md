# ASSCAT Campus AI Copilot Pitch

## Problem Statement
The old student portal is confusing because it uses too many menus. Students waste time searching for information like schedules, clearance, and announcements.

## Industry 5.0 Justification
The AI Copilot helps students by giving fast and smart answers. It supports students instead of replacing them by guiding them in their tasks.

## User Stories
1. As a student, I want to ask the AI about my schedule, so that I can quickly see my classes.
2. As a student, I want to check my clearance using AI, so that I don’t need to navigate many menus.
3. As a student, I want to receive announcements from AI, so that I stay updated easily.
